# AJAX Test

## 一、介绍

本项目为学习 AJAX 时进行的代码测试片段。使用 AJAX 响应 html、css、js、json、xml。

- server.js —— Node.js 代码
- main.js —— AJAX 主代码
- index.html —— 按钮控制主页面
- 响应文件
  1. style.css
  2. add.html
  3. index.js
  4. add2.xml
